# 知识点讲解
1. 二分查找的思想（猜价格游戏、「力扣」第 704 题：二分查找、把区间分成三个部分）；
2. 为什么取中间（取其它地方可不可以。整数除法带来的问题：数组长度为偶数的时候和奇数的时候）；
3. 把区间分成两个部分：一定不存在目标元素的区间，可能存在目标元素的区间；
4. 区间划分与中间数取法的配对关系（一定要举例，强调在区间里只剩下两个元素的时候）；
5. 二分查找算法的应用（在有序数组中查找元素、查找一个有范围的整数）。
