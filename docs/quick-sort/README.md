## 归并排序与快速排序都使用了「分治思想」

### 归并排序

- 拆分：不管数组的形态，总是将数组一分为二；
- 组合：合并两个有序的数组。

### 快速排序

- 拆分：根据某个元素 `pivot`，将数组整理成两个部分；

  - 前半部分小于 `pivot`， 后半部分大于等于 `pivot`；
  - 把 `pivot` 交换到前半部分的最后一个元素。

- 组合：什么都不用做。
